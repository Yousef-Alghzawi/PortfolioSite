# Yousef Alghzawi – Portfolio

Professional portfolio for Yousef Alghzawi, Clinical Biostatistician.

## Features
- **Modern Dark Theme:** Custom design using Tailwind CSS and Space Mono.
- **Interactive Background:** Hero section features a custom canvas-based mathematical animation.
- **Responsive Layout:** Sidebar navigation on desktop, sticky header on mobile.
- **Deep-Dive Case Studies:** Detailed breakdown of statistical methods, outcomes, and reproducible code.
- **Seamless Transitions:** Uses View Transitions API for smooth page navigation.

## Infrastructure (GitHub Pages)
This repository is optimized for hosting on **GitHub Pages**:
- `.nojekyll`: Disables Jekyll processing for faster, cleaner builds.
- `404.html`: Custom error page matching the site theme.
- `sitemap.xml` & `robots.txt`: Search engine optimization.
- `.github/workflows/deploy.yml`: Automated deployment pipeline.

## Development
To run this site locally:
1. Clone the repository.
2. Use any static file server (e.g., `npx serve .` or `python -m http.server`).
3. Open `localhost` in your browser.

## Deployment
Changes pushed to the `main` branch are automatically deployed to GitHub Pages via the included GitHub Action.
