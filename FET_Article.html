Meta-Analysis of Proportion Outcomes: Stent vs. FET

Proportion Pooling • Freeman-Tukey PFT • Continuous Outcomes • Publication Bias

metaprop() Freeman-Tukey PFT metamean() REML Trim-and-Fill Egger Regression

Problem

Pooling event rates and proportions across studies comparing stent grafting versus frozen elephant trunk in Type A aortic dissection. Studies varied in sample size and reported outcomes, with sparse events requiring specialized methods.

Constraints

Zero-event studies requiring appropriate handling. Varying follow-up durations. Single-arm nature of some studies. Need for variance stabilization for proportion pooling.

Methodological Approach

Generalized linear mixed models for proportion pooling with Freeman-Tukey double arcsine transformation. Peto's method for rare events. PRISMA-compliant systematic review with comprehensive heterogeneity assessment.

Output

  • • Forest plots for pooled proportions
  • • Funnel plots with asymmetry tests
  • • Subgroup analyses by technique
  • • Complete PRISMA documentation
  • • Reproducible analysis code

Outcome

Published in The Thoracic and Cardiovascular Surgeon. The analysis provided comparative effectiveness data for surgical decision-making. Methodology section required no revision during peer review.

Data Preparation & Proportion Meta-Analysis

# ── Libraries ───────────────────────────────────────────────
library(readxl)
library(dplyr)
library(meta)
library(metafor)

# ── Load extracted data ────────────────────────────────────
dat <- read_excel("data/study_extract.xlsx", sheet = "Main")

# ── Prepare proportion outcome: Approach A ─────────────────
prop_a <- dat %>%
  filter(arm == "A", !is.na(outcome_1_event), !is.na(outcome_1_total)) %>%
  mutate(
    events = as.integer(outcome_1_event),
    total  = as.integer(outcome_1_total),
    label  = paste0(year, " – ", study_id)
  ) %>%
  arrange(year)

# ── Proportion pooling with Freeman-Tukey PFT ──────────────
meta_prop_a <- metaprop(
  event = prop_a$events,
  n     = prop_a$total,
  studlab = prop_a$label,
  sm    = "PFT",        # Freeman-Tukey double arcsine
  method.ci = "CP",     # Clopper-Pearson exact CI
  method.tau = "DL",    # DerSimonian-Laird τ²
  random = TRUE,
  fixed  = FALSE,
  title  = "Outcome 1 – Approach A"
)
summary(meta_prop_a)

# ── Proportion pooling: Approach B ─────────────────────────
prop_b <- dat %>%
  filter(arm == "B", !is.na(outcome_1_event), !is.na(outcome_1_total)) %>%
  mutate(
    events = as.integer(outcome_1_event),
    total  = as.integer(outcome_1_total),
    label  = paste0(year, " – ", study_id)
  ) %>%
  arrange(year)

meta_prop_b <- metaprop(
  event = prop_b$events,
  n     = prop_b$total,
  studlab = prop_b$label,
  sm    = "PFT",
  method.ci = "CP",
  method.tau = "DL",
  random = TRUE,
  fixed  = FALSE,
  title  = "Outcome 1 – Approach B"
)

# ── Continuous outcome: metamean() with MRAW ───────────────
cont_a <- dat %>%
  filter(arm == "A", !is.na(mean_val), !is.na(sd_val)) %>%
  mutate(label = paste0(year, " – ", study_id))

meta_mean_a <- metamean(
  n    = cont_a$n,
  mean = cont_a$mean_val,
  sd   = cont_a$sd_val,
  studlab = cont_a$label,
  sm   = "MRAW",
  method.tau = "REML",
  random = TRUE,
  fixed  = FALSE,
  title  = "Continuous Outcome – Approach A"
)
summary(meta_mean_a)

Publication Bias & Sensitivity

# ── Funnel plot: Approach A proportion ──────────────────────
png("results/funnel_prop_a.png", width = 1200, height = 900, res = 140)
funnel(meta_prop_a,
       xlab = "Proportion (Freeman-Tukey)",
       studlab = TRUE,
       cex.studlab = 0.7)
dev.off()

# ── Egger regression test for funnel asymmetry ─────────────
# Convert metaprop to metafor rma object for Egger test
rma_prop_a <- rma(
  yi = meta_prop_a$TE,
  sei = meta_prop_a$seTE,
  method = "DL"
)

egger_result <- regtest(rma_prop_a, model = "lm")
cat("Egger regression test:\n")
cat("  z =", egger_result$zval, "\n")
cat("  p =", egger_result$pval, "\n")

# ── Trim-and-fill analysis ────────────────────────────────
tf_result <- trimfill(rma_prop_a)
cat("\nTrim-and-fill:\n")
cat("  Imputed studies:", tf_result$k0, "\n")
cat("  Adjusted estimate:", tf_result$beta, "\n")
cat("  Original estimate:", rma_prop_a$beta, "\n")

png("results/funnel_trimfill_a.png", width = 1200, height = 900, res = 140)
funnel(tf_result,
       xlab = "Proportion (Freeman-Tukey)",
       main = "Trim-and-Fill Funnel Plot")
dev.off()

# ── Funnel plot: Approach B proportion ─────────────────────
png("results/funnel_prop_b.png", width = 1200, height = 900, res = 140)
funnel(meta_prop_b,
       xlab = "Proportion (Freeman-Tukey)",
       studlab = TRUE,
       cex.studlab = 0.7)
dev.off()

rma_prop_b <- rma(
  yi = meta_prop_b$TE,
  sei = meta_prop_b$seTE,
  method = "DL"
)

egger_b <- regtest(rma_prop_b, model = "lm")
cat("\nApproach B – Egger test:\n")
cat("  z =", egger_b$zval, "  p =", egger_b$pval, "\n")

tf_b <- trimfill(rma_prop_b)
cat("  Trim-and-fill imputed:", tf_b$k0, "\n")
